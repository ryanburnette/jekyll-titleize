# jekyll-titleize

[Titleize](https://github.com/granth/titleize) as a Liquid Filter in [Jekyll](http://jekyllrb.com).

## Installation

### Gem

Add to `Gemfile`.

```ruby
# Gemfile
gem "jekyll-titleize"
```

And reference in `_config.yml`.

```ruby
# _config.yml
gems: ['jekyll-titleize']
```

### Plugin

Place `jekyll-titleize.rb` into your `_plugins` directory.
